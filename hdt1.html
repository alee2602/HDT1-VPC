a. Calcule la distancia entre estos dos colores en el espacio RGB

b. Convierta ambos colores al espacio HSV (asuma rangos normalizados H[0,1],S[0,1],V[0,1] para simplificar, sabiendo que el Hue del rojo es 0).

d. Argumente matemticamente por qu un algoritmo de agrupacin (clustering) simple fallara en RGB pero funcionara en HSV para determinar que ambos pxeles pertenecen al mismo objeto "pelota roja".

  • En RGB, la separacin numrica entre los dos pxeles es muy grande porque el canal rojo pasa de 50 a 205. Esa diferencia de 205 unidades representa casi la mitad del rango total posible en RGB, as que matemticamente ambos puntos quedan muy lejos uno del otro en el espacio de color.

    Para un algoritmo de clustering, esa distancia tan grande indica que pertenecen a grupos distintos. En cambio, cuando se convierten a HSV, ambos pxeles conservan exactamente el mismo matiz y la misma saturacin, por lo que su distancia en esas dos dimensiones es literalmente cero. La nica diferencia aparece en el valor de brillo, donde uno est en el mximo y el otro alrededor del 20%, lo cual modifica solo una dimensin del espacio HSV sin alterar la posicin del color real.

    Por eso, desde un punto de vista matemtico, en HSV los pxeles quedan mucho ms cercanos y un algoritmo de agrupacin puede reconocerlos como el mismo objeto pelota roja pese al cambio de iluminacin.

def manual_contrast_brightness(image, alpha, beta):
    """
    Aplica g(x) = alpha * f(x) + beta de forma segura.
    Args:
        image: numpy array uint8
        alpha: float (contraste)
        beta: float (brillo)
    Returns:
        numpy array uint8
    """
    # RETO 1: Implementar pipeline seguro
    # 1. Convertir a float32 y normalizar a [0, 1]
    # 2. Aplicar frmula matemtica vectorizada (Sin bucles for)
    # 3. Aplicar np.clip para evitar valores fuera de rango
    # 4. Des-normalizar (x255) y castear a uint8
    
    # TODO: Escribir cdigo aqu
    img = image.astype(np.float32) / 255.0
    img = alpha * img + (beta / 255.0)
    img = np.clip(img, 0.0, 1.0)
    processed_img = (img * 255.0).astype(np.uint8)

def manual_gamma_correction(image, gamma):
    """
    Aplica V_out = V_in ^ gamma
    """
    # RETO 2: Implementar correccin Gamma
    # Recordar: La operacin potencia es costosa. 
    # Usar Look-Up Table (LUT) es una optimizacin comn, pero aqu usa matemticas directas en float.
    
    # TODO: Escribir cdigo aqu
    img = image.astype(np.float32) / 255.0
    img = np.power(img, gamma)
    img = np.clip(img, 0.0, 1.0)
    gamma_img = (img * 255.0).astype(np.uint8)
    return gamma_img

def hsv_segmentation(image):
    """
    Segmentar un objeto de color especfico (ej. verde o rojo)
    """
    # 1. Convertir a HSV
    hsv = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)
    
    # RETO 3: Definir rangos para un color.
    # OJO: En OpenCV Hue es [0, 179].
    # Ejemplo: Si buscas verde, H est alrededor de 60 (en escala 0-179).
    # TODO: Definir lower_bound y upper_bound (np.array)
    # lower_bound = np.array([0, 100, 100])  Este rango es para la imagen carss si se quiere probar, se necesita otro filtro para quitar el amarillo (Creo)
    lower_bound = np.array([35, 20, 20])
    upper_bound = np.array([85, 255, 255])
    
    # Crear mscara
    mask = cv2.inRange(hsv, lower_bound, upper_bound)
    
    # Aplicar mscara a la imagen original (bitwise_and)
    result = cv2.bitwise_and(image, image, mask=mask)
    return result
